# ViewPCQM

Official code of "Information Exploration of Projected Views for  Point Cloud Quality Measurement"


## Get Started

```shell
pip install -r requirements.txt
```

then modify the --train_info, --val_info, --image_dir, and execute 

```shell
python main.py
```
